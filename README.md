SWESree

SWESree is a modern Windows desktop application designed to streamline the repetitive asset optimization tasks involved in professional web design and creative development. Built around a set of carefully refined Python workflows, it provides an elegant drag-and-drop interface for processing images and videos without the complexity of command-line tools.

The application consolidates three essential utilities into a single cohesive experience:

Batch Image → Optimized WebP Conversion
MOV / Video → High-Efficiency WebM Compression
Video → High-Quality Frame Extraction

Rather than replacing the underlying processing logic, SWESree wraps proven, production-tested Python workflows inside a premium desktop interface, preserving carefully tuned defaults while exposing advanced controls for users who need deeper customization.

Key Features
✨ Beautiful native Windows desktop experience.
📂 Drag-and-drop support for files and entire folders.
⚡ One-click processing with intelligent defaults.
🎛️ Advanced controls with optional parameter overrides.
📁 Automatic organized output folders to keep source directories clean.
🖼️ Batch conversion of images to optimized WebP format.
🎥 High-efficiency WebM video compression powered by FFmpeg.
🎞️ Video frame extraction with automatic folder management.
📊 Live progress tracking, status updates, and activity logs.
🧵 Background processing for a responsive, non-blocking interface.
🔧 Built with Python and PySide6 for maintainability and future extensibility.
